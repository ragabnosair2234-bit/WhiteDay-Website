# دليل المطور (Developer Guide) - White Day Project

هذا الدليل مخصص لأي مطور سيعمل على المشروع مستقبلاً بغرض إضافة صفحات جديدة، خدمات إضافية، أو التعديل على تصميم الموقع بشكل عام.

## 📁 هيكل المشروع (Project Structure)
المشروع مبني باستخدام **HTML** المتجاوب بدعم من إطار عمل **TailwindCSS** (عن طريق الـ CDN) بالإضافة لملف أصلي **style.css** لبعض التخصيصات والألوان الثابتة.

```text
D:\3laa\
├── index.html           # الصفحة الرئيسية (Landing Page)
├── style.css            # ملف التنسيق الأساسي (الخطوط، لون الزر الرئيسي، وحجم اللوجو)
├── img/                 # مجلد يحتوي على جميع الصور المستخدمة في الموقع
└── Pages/               # مجلد يحتوي على جميع الصفحات الداخلية والخدمات
    ├── welcome.html     # صفحة الخدمات الشاملة التي يتم تحويل المستخدم إليها
    ├── login.html       # صفحة تسجيل الدخول
    ├── signup.html      # صفحة إنشاء الحساب
    ├── halls.html       # صفحة نماذج الخدمة (مثال: القاعات)
    ├── cars.html        # صفحة السيارات
    └── ... وغيرها من الخدمات
```

## 🛠️ كيف تضيف صفحة خدمة جديدة؟ (How to add a new service page)

إذا أردت مثلاً إضافة صفحة جديدة لخدمة **Catering**، اتبع الخطوات التالية بالترتيب:

### 1️⃣ إنشاء صفحة الخدمة (Create the Page)
أسهل طريقة للبدء هي نسخ صفحة موجودة بالفعل لتคงافظ على التصميم العام للموقع.
1. قم بعمل نسخة (Copy/Paste) لملف `halls.html` (أو `cars.html`) الموجود داخل مجلد `Pages/`.
2. قم بتسمية النسخة الجديدة بالاسم المناسب، مثلاً: `catering.html`.

### 2️⃣ تعديل المحتوى (Edit Content)
افتح `catering.html` في الـ Editor وقم بالتعديلات الآتية:
- في بداية الملف، غير العنوان الرئيسي في الترويسة: 
  ```html
  <h2 class="text-4xl md:text-5xl font-bold italic-text mt-4">The Catering</h2>
  ```
- قم بتكرار أو إزالة الكروت (Cards) الموجودة في قالب الـ `grid` بحسب عدد مقدمي الخدمة.
- تأكد من تعديل الصورة `src="../img/your-image.jpg"`.
- قم بتعديل اسم مقدم الخدمة والسعر.

### 3️⃣ ربط الصفحة في Welcome (Linking in Welcome)
1. افتح صفحة `Pages/welcome.html`.
2. ابحث عن الأزرار الخاصة بالخدمات.
3. إذا كان هناك زر يحمل مساراً فارغاً (مثلاً `<a href="#">Book a catering</a>`)، قم بتعديله ليكون:
   ```html
   <a href="./catering.html" class="btn-primary btn-category w-auto px-8">Book a catering</a>
   ```

### 4️⃣ ربط الصفحة في الـ Index (اختياري)
إذا كنت تريد وضع كارت بارز لهذه الخدمة في الصفحة الرئيسية (`index.html`):
1. افتح `index.html`.
2. اذهب إلى قسم **Services Showcase** (ستجده تحت Hero Section).
3. أضف الكود التالي داخل قسم الـ Grid:
   ```html
   <div class="text-center">
       <h3 class="font-serif italic font-bold text-2xl mb-4">Catering</h3>
       <!-- للربط المباشر إذا أردت يمكن إحاطة الصورة بـ <a> tag -->
       <img src="./img/your-catering-image.jpg" alt="Catering" class="w-full h-[300px] md:h-[400px] object-cover rounded-3xl shadow-md">
   </div>
   ```

## 🎨 التحكم في الألوان والتصميم (Color & Styling)

### اللون الرئيسي (Brand Color)
لتغيير اللون الأساسي الموحد للموقع (لون الأزرار، التظليلات، وبعض الفواصل)، قم بفتح `style.css` وغير قيمة متغير `--MainColor`:
```css
:root {
  --MainColor: #D4AEA8; /* لتغيير لون الأزرار غير قيمة الـ Hex هنا فقط */
}
```
*جميع أزرار الموقع في مختلف اللغات تأخذ لونها تلقائياً من هذا المتغير بفضل كلاس `btn-primary`.*

### تصميم الحقول (Forms)
حقول التسجيل (Login/Signup) مُصممة لتمتد لمساحة الشاشة الجانبية. إذا أردت تعديل عرضها أبحث عن الكلاس `max-w-[95%]` داخل ملفات التسجيل وقم بتعديله حسب الرغبة.

## 💡 نصائح عامة (Best Practices)
1. **صور المتصفح:** تأكد من ضغط الصور بحجم مناسب وتسميتها بالإنجليزية كـ `service-name.jpg` ووضعها دائماً في ملف `img/` لتجنب كسر الروابط.
2. **الـ TailwindCSS:** الموقع لا يحتاج عملية البنية (Build Process) لأنه يستخدم الـ CDN الخاصة بـ Tailwind. أي كلاس جديد ستضيفه (مثل `mt-5`, `bg-red-500`) سيتم تطبيقه فوراً.
